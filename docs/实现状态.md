**最后更新**：2026-07-01

> 本文是**实现边界的唯一权威来源**。简历与面试只写本文「已实现」部分。

## 运行时模型

```text
main (SIGINT/SIGTERM 优雅退出)
  └── gateway_app
        ├── ingress_thread  → modbus_adapter.poll → SPSC ring
        ├── worker_thread   → SQLite WAL → state_engine → command_scheduler → MQTT
        └── monitor_thread  → epoll/timerfd → watchdog / 心跳 / MQTT 补传 / PING
```
